.. _features-and-functionalities-1:

**Features and Functionalities**
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

The developer tab includes most of the features developed for the
platform users specially data engineers, data scientists, ML engineers
etc. It is a one stop option for the users to access data from Data Lake
Explorer, analyze the data in Auto-EDA, write experiments in Notebooks,
understand and play with the results in Models and Experiments, and much
more.
